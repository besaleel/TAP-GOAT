# Tap Goat

Jogo casual para celulares e tablets: uma Cabra Maratimba fica andando em um pasto e, ao tocar nela, o jogador ouve um som de balido engraçado e vê a cabra fazer uma careta antes de voltar a andar. O objetivo é simples: entreter o jogador através da interação com a cabra.

- **Nome na loja:** Tap Goat
- **Pacote (applicationId):** `com.tapgoat.app`

## Tecnologias

- HTML5, CSS3, JavaScript
- SVG para animações
- Three.js para a cena 3D da cabra
- AngularJS, IONIC e Capacitor para o app webview (Android)

## Premissas

- Desenvolvido para Android, com suporte a diferentes tamanhos de tela.
- O jogo deve funcionar **100% offline**, sem dependência de bibliotecas ou fontes externas — tudo (imagens, sons, fontes) é embutido no aplicativo.
